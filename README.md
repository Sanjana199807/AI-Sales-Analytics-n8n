# AI-Sales-Analytics-n8n
AI-powered backend code generation using LLM &amp; workflow automation
**Project Overview**

This project demonstrates an AI-powered analytics workflow that allows users to query business data using natural language. The system converts user questions into SQL queries, retrieves the relevant data from a database, and generates insights along with visualizations.

The workflow automation is implemented using n8n and the database operations are handled using PostgreSQL.

The goal of this project is to simplify business analytics by allowing users to interact with data without requiring SQL knowledge.

**Key Features**

Natural language query input

Automatic SQL query generation

Database integration with PostgreSQL

Automated data retrieval

AI-generated insights

Chart visualization for analytical results

**Technologies Used**

n8n – Workflow automation platform

PostgreSQL – Relational database

SQL – Data querying language

AI Language Model – Natural language understanding and query generation

**System Workflow**
The project follows an automated workflow pipeline:

User enters a natural language query

AI interprets the query

SQL query is generated automatically

Query is executed in PostgreSQL

Data is retrieved and processed

Charts and insights are generated

Results are displayed to the user

**Example Query**

Example user prompt:

Compare total sales for New vs Returning customers across regions

Generated analysis example:

East region shows higher sales from returning customers

North region shows stronger sales from new customers

West region has the highest contribution from returning customers

This demonstrates how AI can generate insights automatically from database queries.
